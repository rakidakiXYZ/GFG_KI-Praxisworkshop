# Iteratives Verfeinern: Die Kunst des Nachpromptens

## Wozu dient diese Technik?

Die meisten Menschen behandeln ChatGPT wie einen Automaten: Prompt rein, Ergebnis raus, fertig. Wenn das Ergebnis nicht passt, fangen sie komplett von vorne an.

Die Wahrheit: **Der erste Output ist nie der finale Output.** Die besten Ergebnisse entstehen nicht aus dem perfekten Prompt, sondern aus **mehreren Runden gezielten Nachpromptens**. Sie formen das Ergebnis Schritt für Schritt — wie ein Bildhauer, der aus einem groben Block ein Kunstwerk meißelt.

## Das Prinzip

```
Erster Prompt → Rohversion (70% gut)
    ↓ Feedback
Iteration 1 → Hook verbessert (80% gut)
    ↓ Feedback
Iteration 2 → Mittelteil gestrafft (90% gut)
    ↓ Feedback
Iteration 3 → CTA geschärft (95% gut) ✅
```

Jede Runde adressiert **ein spezifisches Problem** — nicht alles auf einmal.

## So verwenden Sie die Technik

1. **Erster Prompt:** Formulieren Sie Ihre Aufgabe so gut Sie können. Akzeptieren Sie, dass das Ergebnis nicht perfekt sein wird.
2. **Lesen, nicht scannen:** Lesen Sie das Ergebnis sorgfältig. Was funktioniert? Was stört?
3. **Ein Problem pro Iteration:** Benennen Sie genau, was nicht stimmt — und nur das. Nicht: „Mach alles besser." Sondern: „Der Einstieg ist zu schwach."
4. **Loben, was funktioniert:** „Das ist gut, ABER..." funktioniert besser als nur Kritik. Die KI behält dann die guten Teile bei.
5. **Wiederholen** bis Sie zufrieden sind (typisch: 2–4 Runden).

### 💡 Tipps

- **Nie komplett neu starten:** Wenn 70% des Ergebnisses gut sind, werfen Sie es nicht weg. Verfeinern Sie die 30%, die nicht passen.
- **Spezifisch sein:** „Mach es besser" ist nutzlos. „Ersetze den ersten Satz durch eine provokante Frage" ist präzise.
- **Referenzieren:** Sagen Sie *„In Absatz 3..."* oder *„Die zweite Aufzählung..."* — damit die KI weiß, wo sie ansetzen soll.
- **Vergleich fordern:** *„Gib mir 2 Alternativen für den Einstieg"* — dann wählen Sie die beste.
- **Funktioniert auch für Code:** „Die Funktion ist korrekt, aber refactore sie für bessere Lesbarkeit" — iteratives Verfeinern ist beim Programmieren besonders effektiv.

## Die wichtigsten Nachprompts

### Hook / Einstieg verbessern

```
Das ist gut, aber der Einstieg ist zu schwach.
Schreibe die ersten 2 Sätze neu:
- Nutze eine provokante Frage ODER eine überraschende Zahl
- Der Leser muss sofort denken: "Das will ich wissen"
- Behalte den Rest unverändert bei
```

### Kürzen und straffen

```
Der Mittelteil ist zu lang und wiederholt sich.
Kürze ihn um 40%, ohne Inhalt zu verlieren.
Jeder Satz muss seinen Platz verdienen.
Streiche Fluff, Füllwörter und Wiederholungen.
```

### Konkreter machen

```
Das ist zu abstrakt. Ersetze jede generische Aussage
durch ein konkretes Beispiel.
Statt "KI kann im Bankwesen helfen" →
"Eine Volksbank nutzt KI, um Kreditanträge in 5 statt 20 Minuten zu prüfen."
```

### Ton anpassen

```
Der Text klingt zu formell / zu locker / zu generisch.
Schreibe ihn um in einem Ton, der [GEWÜNSCHTER TON] ist.
Denk an [VORBILD — z.B. "wie ein TED-Talk" / "wie eine WhatsApp
an einen Kollegen" / "wie ein Zeitungsartikel"].
```

### Call-to-Action schärfen

```
Der Schluss verpufft. Schreibe das Ende neu:
- Klarer Call-to-Action: Was soll der Leser/Zuschauer JETZT tun?
- Nur EINE Handlung (nicht drei Optionen)
- Dringlichkeit erzeugen, ohne aufdringlich zu sein
```

### Alternativen generieren

```
Gib mir 3 alternative Versionen von [ABSCHNITT / SATZ / ÜBERSCHRIFT].
Jede soll einen anderen Ansatz verfolgen:
1. Emotionaler
2. Datengetriebener
3. Provokanter
```

## Häufige Fehler beim Nachprompten

| ❌ Fehler | ✅ Besser |
|----------|----------|
| „Mach es besser" | „Der Hook ist zu schwach — starte mit einer Frage" |
| „Das gefällt mir nicht" | „Absatz 2 ist zu lang und wiederholt Absatz 1" |
| „Nochmal von vorne" | „Behalte die Struktur, überarbeite nur den Ton" |
| „Mehr Details" | „Füge in Abschnitt 3 ein konkretes Zahlenbeispiel ein" |
| Alles auf einmal ändern | Ein Problem pro Iteration |

---

> 💡 **Die Denkweise:** Behandeln Sie ChatGPT nicht wie einen Automaten, sondern wie einen **Mitarbeiter im Entwurfsprozess**. Erster Entwurf → Feedback → Überarbeitung → Feedback → Finale Version. Genau so arbeiten Profis mit Textern, Designern und Agenturen — und genau so holen Sie das Maximum aus der KI.
