# Professionelles LinkedIn-Portraitfoto aus einem Selfie erstellen

## Wozu dient dieser Prompt?

Ein professionelles Bewerbungs- oder LinkedIn-Foto kostet beim Fotografen schnell 100–300 €. Mit diesem Prompt verwandeln Sie ein einfaches Foto von sich in ein hochwertiges Business-Portrait — direkt in ChatGPT oder Google Gemini.

## So verwenden Sie den Prompt

1. **Foto vorbereiten:** Nehmen Sie ein gut belichtetes Foto von sich (Smartphone reicht). Wichtig: Gesicht klar erkennbar, frontal oder leicht seitlich.
2. **Prompt + Foto zusammen senden:** Öffnen Sie ChatGPT (mit GPT-4o oder höher) oder Google Gemini. Laden Sie Ihr Foto hoch und fügen Sie den Prompt darunter ein.
3. **Ergebnis prüfen:** Die KI generiert ein professionelles Portrait basierend auf Ihrem Aussehen. Prüfen Sie, ob Gesichtszüge und Proportionen stimmen.
4. **Anpassen:** Sie können Farben, Kleidung oder Hintergrund im Prompt ändern — z.B. einen schwarzen Blazer statt blau, oder einen neutralgrauen Hintergrund.

### 💡 Tipps

- **Fotografie-Begriffe im Prompt** (wie „85 mm Festbrennweite" oder „f/2.0") müssen Sie nicht verstehen. Das sind Anweisungen für die KI, damit das Ergebnis wie ein echtes Studiofoto aussieht. Sie kopieren den Prompt einfach so wie er ist.
- **Brille, Bart, Frisur** — die KI übernimmt diese Details aus Ihrem Foto. Verwenden Sie ein aktuelles Bild.
- **Andere Tools:** Der Prompt ist für ChatGPT und Gemini optimiert. In Midjourney oder DALL-E direkt funktioniert er ebenfalls, aber das Ergebnis kann abweichen — dort fehlt die Foto-Analyse.
- **Mehrere Varianten:** Lassen Sie 2–3 Versionen generieren und wählen Sie die beste.
- **Format:** Das Ergebnis ist quadratisch (1:1) — perfekt für LinkedIn, Xing und Teams-Profilbilder.

> ⚠️ **Hinweis:** KI-generierte Portraits können leichte Abweichungen bei Gesichtszügen zeigen. Das Ergebnis eignet sich hervorragend als professionelles Profilbild, sollte aber nicht als Passfoto oder für offizielle Dokumente verwendet werden.

## Der Prompt

```
Professionelles, hochauflösendes Portraitfoto im Querformat 1:1 (square)
als Headshot von derselben Person wie im hochgeladenen Foto —
mit identischer Gesichtsstruktur, Hautton und klar erkennbarer Identität.

<framing>
Brustaufwärts im Bild, mit etwas Luft über dem Kopf.
Direkter, selbstbewusster Blick in die Kamera.
Wirkung: wie mit einer 85 mm Festbrennweite bei f/2.0 fotografiert —
leichte Schärfentiefe, Fokus auf den Augen.
</framing>

<clothing>
Hochwertiger, moderner, blauer Smart-Casual-Blazer.
Kein Krawatte, kein Einstecktuch — clean und zeitgemäß.
</clothing>

<lighting>
Weiches, helles Frontlicht (Beauty Dish oder große Softbox)
für ein gleichmäßig aufgehelltes Gesicht ohne harte Schatten.
Dezentes Kanten- bzw. Haarlicht von hinten/seitlich,
das die Konturen von Kopf und Schultern klar vom Hintergrund trennt.
</lighting>

<background>
Einfarbig, dunkles Rotbraun (#562226).
Weich unscharf (Bokeh), etwas dunkler als das Gesicht.
Guter Helligkeits- und Farbkontrast zur Person.
</background>

<mood>
Selbstbewusst, professionell und zugleich nahbar.
Moderne, leicht warme Farbwirkung.
Fotostudio-Qualität, kein KI-Look.
</mood>
```
