# Prompt Templates: Wiederverwendbare Vorlagen bauen lassen

## Wozu dient diese Technik?

Sie schreiben regelmäßig E-Mails, Social-Media-Posts, Berichte oder Zusammenfassungen? Dann tippen Sie wahrscheinlich jedes Mal ähnliche Anweisungen in ChatGPT — Ton, Länge, Format, was vermieden werden soll. Das ist wie jeden Morgen die Kaffeemaschine neu erklären.

Die Lösung: Lassen Sie ChatGPT **einmalig eine Vorlage für Sie bauen**, die Sie danach immer wieder verwenden. Das Besondere: Sie schreiben die Vorlage nicht selbst — ChatGPT interviewt Sie und baut sie basierend auf Ihren Antworten.

## So verwenden Sie die Technik

1. **Aufgabe identifizieren:** Was machen Sie regelmäßig? (E-Mails, Posts, Berichte, Skripte…)
2. **Interview starten:** Kopieren Sie den Prompt unten und lassen Sie sich befragen.
3. **Vorlage erhalten:** ChatGPT baut Ihnen eine maßgeschneiderte Vorlage.
4. **Vorlage speichern:** Kopieren Sie die fertige Vorlage in eine Textdatei, Notiz-App oder ein ChatGPT-Projekt.
5. **Wiederverwenden:** Bei jeder neuen Aufgabe: Vorlage einfügen + Details ergänzen → fertig.

### 💡 Tipps

- **Eine Vorlage pro Aufgabentyp:** E-Mail-Vorlage, Social-Media-Vorlage, Berichts-Vorlage — nicht alles in eine packen.
- **Vorlage iterieren:** Nach 5–10 Nutzungen merken Sie, was fehlt oder stört. Passen Sie die Vorlage an.
- **In Projects speichern:** Legen Sie ein ChatGPT-Projekt „Meine Vorlagen" an und speichern Sie alle Templates dort. So hat ChatGPT sie immer parat.
- **Außerhalb von ChatGPT:** Speichern Sie Vorlagen auch in einer Notiz-App (Apple Notizen, Google Keep, OneNote) in einem Ordner „KI-Vorlagen" — so finden Sie sie auch ohne ChatGPT.
- **Versionieren:** Speichern Sie Vorlagen mit Datum (z.B. „E-Mail-Template-v3-März2026"). So sehen Sie, wie sich Ihre Vorlagen über die Zeit verbessern.
- **Upfront-Aufwand, langfristiger Gewinn:** 10 Minuten Vorlage bauen spart Ihnen hunderte Male jeweils 2–3 Minuten.

## Der Prompt: Vorlage bauen lassen

```
<aufgabe>
Hilf mir, eine wiederverwendbare Prompt-Vorlage zu erstellen
für folgende Aufgabe: [AUFGABENTYP — z.B. "professionelle E-Mails
schreiben" / "Social-Media-Posts erstellen" / "Meeting-Zusammenfassungen"]
</aufgabe>

<interview>
Stelle mir Fragen, um die Vorlage perfekt auf mich zuzuschneiden:
- Welchen Ton verwende ich normalerweise?
- Welche Elemente müssen immer enthalten sein?
- Was soll vermieden werden?
- Wie lang soll das Ergebnis typischerweise sein?
- Gibt es Formulierungen oder Phrasen, die ich mag oder hasse?
- Für wen schreibe ich das meistens?

Stelle die Fragen nacheinander, nicht alle auf einmal.
</interview>

<ergebnis>
Erstelle danach eine fertige Prompt-Vorlage, die ich
kopieren und immer wieder verwenden kann.

Die Vorlage soll:
- Platzhalter in [ECKIGEN KLAMMERN] haben für variable Inhalte
- Alle meine Präferenzen fest eingebaut haben
- Selbsterklärend sein (ich will sie in 6 Monaten noch verstehen)
- In einem Code-Block stehen, damit ich sie leicht kopieren kann
</ergebnis>
```

## Beispiel: Fertige E-Mail-Vorlage

So könnte eine Vorlage aussehen, die ChatGPT für Sie erstellt:

```
Schreibe eine kurze, professionelle E-Mail.

<kontext>
Typ: [outreach / follow-up / anfrage / antwort]
Empfänger: [Name, Rolle, Beziehung]
Anlass: [Worum geht es?]
Gewünschte Reaktion: [Was soll der Empfänger tun?]
</kontext>

<stil>
- Direkt, selbstbewusst, locker-professionell
- Keine Corporate-Floskeln ("Ich hoffe, es geht Ihnen gut")
- Kurze Absätze (1–2 Zeilen max.)
- Klarer Call-to-Action am Ende
- Keine Signatur (füge ich selbst ein)
- Starker Einstieg ohne Smalltalk
</stil>

<format>
- Betreffzeile + E-Mail-Text
- Maximal 100 Wörter
</format>
```

## Welche Vorlagen lohnen sich?

| Aufgabe | Häufigkeit | Zeitersparnis pro Nutzung |
|---------|-----------|--------------------------|
| E-Mails (Outreach, Follow-up) | Täglich | 3–5 Min |
| Social-Media-Posts | Mehrmals/Woche | 5–10 Min |
| Meeting-Zusammenfassungen | Wöchentlich | 10–15 Min |
| Berichte / Analysen | Monatlich | 20–30 Min |
| Video-Skripte | Wöchentlich | 15–20 Min |
| Kunden-Proposals | Nach Bedarf | 30+ Min |

**Faustregel:** Wenn Sie etwas mehr als 3× im Monat machen, lohnt sich eine Vorlage.
